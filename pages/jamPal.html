JamPal

JamPal has been a fun side project that I've been working on with my brother. The mobile app is designed to be a tool for musicians of any style to find common songs to play together. It is written in Flutter, a google-based framework for cross-platform applications (android, apple, web), and we have an Express.js back-end service for user authentication and song management. It is mostly a local application, using an sqlite3 database for song storage.
