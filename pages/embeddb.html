EmbedDB and Thesis

I had the great pleasure to do a graduating thesis under the supervision of Dr. Ramon Lawrence. The thesis was focused on adding improvements to EmbedDB, a lightweight database system designed to be run on low performance hardware. EmbedDB is designed to run with only 4 kilobytes of memory and doesn't need and operating system, which allows it to run on devices that have very little memory. I was tasked with fixing the Order By functionality and various other tests and project issues from previous development by other students.

You can view the Github repository here.

The fixes were largely focused on memory management problems during the sorting methods, with some final results comparing performance to SQLite on different hardware. The completion of Order By allows for data analysis on low-performance devices like Arduinos that lack an operating system by connecting the existing Group By functionality.

This work provided an immense opportunity to grow my understanding of low-level systems and memory management in C, and getting the chance to work with PlatformIO on different embedded devices, mainly the Arduino Due. The challenges of cross-platform development was a fantastic experience for my skills.

My thesis can be viewed here at the UBC Library.
